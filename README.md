# 📖 Descrição do Projeto

O **Sabor Express** é uma aplicação desenvolvida em **Python** durante o curso **"Python: aplicando a Orientação a Objetos"**, da Alura.

O projeto tem como objetivo apresentar, na prática, os principais conceitos da **Programação Orientada a Objetos (POO)** utilizando Python, aplicando boas práticas de modelagem de classes, encapsulamento e organização do código.

Durante o desenvolvimento da aplicação foram implementados conceitos como:

* Classes e Objetos;
* Métodos de Instância;
* Métodos de Classe (`@classmethod`);
* Métodos Estáticos (`@staticmethod`);
* Propriedades (`@property`);
* Encapsulamento;
* Construtores (`__init__`);
* Métodos Especiais (`__str__`);
* Organização em módulos e pacotes;
* Relacionamento entre classes;
* Manipulação de listas de objetos.

O sistema simula o gerenciamento de restaurantes, permitindo cadastrar restaurantes, alterar seus estados, registrar avaliações e calcular automaticamente a média das notas recebidas.

# 📂 Acesso ao Projeto

Clone o repositório:

```bash
git clone https://github.com/mathsant-js/oo-sabor-express.git
```

Entre na pasta do projeto:

```bash
cd oo-sabor-express
```

Execute a aplicação:

```bash
python app.py
```
